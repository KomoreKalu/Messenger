# Messenger

使用smstools3转发短信至TG/QQ

### 依赖

`bash curl smstools3 kmod-usb-serial kmod-usb-serial-wwan kmod-usb-serial-option usb-modeswitch`

### 使用

`ID`TG chatid或QQ号

`TOKEN`TG bot token 或 Qmsg token(与上面对应)
